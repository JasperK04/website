# Computer Architecture - LC-3 Assembly

This project collects the final LC-3 assembly assignments from the Computer Architecture course. The course covers binary representations, logic gates and circuits, storage and memory, finite state machines, and the von Neumann model, then applies these ideas to the LC-3 instruction set and assembly programming.

## Course focus

- Binary representation of numbers, text, images, and instructions
- Logic gates, combinational circuits, and storage elements
- Finite State Machines
- The von Neumann model and the LC-3 ISA datapath
- Basic I/O (polling, memory-mapped I/O, interrupts)
- Assembly programming fundamentals and control flow

## Assignments

### RNG (Pseudo-random number generator)

File: `RNG.asm`

Implements a simple pseudo-random number generator using LC-3 operations and control flow.

### Primes (Prime number detection)

File: `primeNum.asm`

Checks for primality using repeated division and loop-based control flow, exercising arithmetic and branching.

### BrainFry (Final project)

File: `BrainFry.asm`

A small interpreter for a Brainfuck-like language, emphasizing character I/O, memory manipulation, and structured control flow in LC-3 assembly.

## Notes

- All three files are LC-3 assembly programs.
- The code is intended to be assembled and run on an LC-3 simulator.
